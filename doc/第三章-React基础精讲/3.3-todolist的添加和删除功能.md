### todolist的添加和删除功能：
```javascript
import React, { Component ,Fragment} from 'react';
class Todolist extends Component {
    constructor(props){
        super(props)
        this.state={
            inputValue:"",
            list:[]
        }
    }
  handleInputChange=(e)=>{
    this.setState({inputValue:e.target.value})
    console.log(this.state.inputValue)
    }
    onSubmit=()=>{
        const list=[...this.state.list,this.state.inputValue]; //1*
        this.setState({
            list,
            inputValue:""
        })
    }
    handleDelete=(index)=>{
        const list=[...this.state.list]; // 1*
        list.splice(index,1);
        this.setState({
            list
        })
    }
    render() { 
        return (
            <Fragment>
                <div>
                    <input onChange={this.handleInputChange} value={this.state.inputValue}/>
                    <button onClick={this.onSubmit}>提交</button>
                </div> 
                <ul>
                    {this.state.list.map((item,index)=>{
                        return <li onClick={this.handleDelete} key={item} >{item}</li>
                    })}
                </ul>
            </Fragment>
            );
        }
    }
export default Todolist;
```
### 注释<br/>
** 1：react中有个immutable的理念，就是不能直接在原来的state上直接修改状态，而是要修改副本，然后通过setstate方法去修改state。
**

**
注意 “...”操作符的使用，尤其像是这种添加、删除的操作啥的，比较方便，另外通过filter方法也可以实现
