## 使用redux-thunk实现ajax数据请求

🌟能够发现在使用redux-thunk过程中一共执行了两次dispatch函数：<br/>
<br/>🐒第一次是在todolist.js中，将creator.js中const action=actions.getDataAction()通过dispatch发给store并自动执行～！！！🌿；<br/>
<br/>（正常来说action.js中的action创建函数都是返回对象的，但是actions.getDataAction却是返回函数，就是因为使用了redux-thunk才能让它得以为所欲为😤）<br/>
<br/>🐱这个自动执行的函数自己就可以接收一个dispatch为参数的方法，就是在actioncreator中的dispatch，然后继续派发action改变store内数据。

import reducer from "./reducer";
import thunk from "redux-thunk";

const composeEnhancers =
  typeof window === 'object' &&
  window.__REDUX_DEVTOOLS_EXTENSION_COMPOSE__ ?   
    window.__REDUX_DEVTOOLS_EXTENSION_COMPOSE__({
    }) : compose;

const enhancer = composeEnhancers(
        applyMiddleware(thunk),
      );
      const store = createStore(reducer, enhancer);

export  default store;
